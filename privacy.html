Privacy Policy

This extension does not collect, store, or share any personally identifiable information from users.

Highlighted text is temporarily sent to the Gemini and OpenAI APIs in order to generate explanations or text-to-speech output. This data is used only for providing the requested functionality and is not stored by the extension.

User preferences (such as API keys or settings) are stored locally using Chrome's storage API and are never transmitted to external servers other than the selected API providers.

By using this extension, you agree that highlighted text may be processed by the Gemini and/or OpenAI APIs for the sole purpose of generating results. No data is sold, shared, or used for advertising.
